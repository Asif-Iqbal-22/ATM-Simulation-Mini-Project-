# ATM Simulation in C

A simple console-based ATM simulation program built using C.

## Features
- PIN login authentication
- Check account balance
- Deposit money
- Withdraw money
- Menu-driven interface

## How to Run
1. Compile the program:

    gcc atm.c -o atm

2. Run:

    ./atm

## Future Improvements
- Multiple user accounts
- Store balance in file
- Better PIN security
